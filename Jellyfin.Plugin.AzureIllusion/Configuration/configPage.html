AzureIllusion

Dokumentacja API

Plugin pobiera polskie napisy z AzureIllusion. Witryna otrzymuje wyłącznie identyfikator AniList.

W produkcji wymagany jest adres HTTPS.
Klucz pozostaje w konfiguracji serwera Jellyfin.
Pozwala zadaniom i bibliotekom Jellyfin korzystać z providera automatycznie.
Lista jest pobierana dynamicznie z AzureIllusion.
0 oznacza wszystkie. Domyślnie wybierana jest jedna najlepsza grupa.
PL2 jest pokazywany w nazwie wyniku, a Jellyfin zapisuje oba warianty jako język polski.
Używane tylko bez AniList ID. Niejednoznaczne wyniki są pomijane.
Przykład: {"anidb:123":456,"kitsu:abc":456}. Te dane nigdy nie są wysyłane do AzureIllusion.
